# Skill: Implement Frontend

> Execute an assigned frontend TODO with reuse checks, Playwright-first validation, and clean handoff.

## Trigger

Activated by `/project:dev-frontend` with a specific TODO assignment.

## Process

1. **Read** — UI Kit (docs/ui/UI_KIT.md) + assigned TODO
2. **Plan** — Component inventory, reuse existing components
3. **Implement** — Diagnostic-first layout, responsive design
4. **Test** — Playwright E2E tests for user-facing flows
5. **Verify** — Check at multiple viewports (1024, 1280, 1920)
6. **Handoff** — Update session state, commit

## Output

- Component/page code + Playwright tests
- Screenshots at key viewports
- Updated UI Kit if new design tokens added
